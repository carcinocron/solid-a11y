`solid-a11y` is a collection of fully accessible, completely unstyled (headless) components for
[SolidJS][solid_js].

## Documentation

For full documentation, visit [www.solid-a11y.dev][doc_site].

## Installation

```sh
$ npm add solid-a11y
```
