# solid-a11y

A collection of fully accessible, completely unstyled (headless) components for
[solid-js][solid_js].

## Documentation

For full documentation, visit [www.solid-a11y.dev][doc_site].

## Installation

```sh
$ npm add solid-a11y
```

## Roadmap

- SSR support
- More components:
  - combobox (autocomplete)
  - listbox (select)
