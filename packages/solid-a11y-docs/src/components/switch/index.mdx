## Labeling

An easy way to add screen reader friendly text is by using [visually hidden text][visually_hidden]
as a direct child of the switch.

<CodeSampleShowcase source={ProcessedWithDirectLabelExample} />

One can use the `<Label />` and/or `<Description />` components to provide associate ARIA compliant
labels to the switch, but still allowing the text to be visible and elsewhere in the DOM.
