## Proper Labeling

One should label their switch with screen-reader friendly text. The easiest way is to add visually
hidden, but not `display: none`, text as a direct child of the switch (for example, a [utility class
like `.sr-only` from Tailwind CSS][tailwind_sr_only]).

<CodeSampleShowcase source={ProcessedWithDirectLabelExample} />

One can use all the `<Label* />` and/or `<Description* />` components to also provideadditional ARIA
support to the switch, while keeping the text visible and elsewhere in the DOM.
